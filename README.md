# GreenDao
greenDAO is a light & fast ORM for Android that maps objects to SQLite databases. Being highly optimized for Android, greenDAO offers great performance and consumes minimal memory.
Home page, documentation, and support links: http://greenrobot.org/greendao/

Introduction
========
* change templates: dao.flt, entity.ftl; add templates: serialize-helper.ftl
* change daogenerator library: add StringList, Class and Enum PropertyType
* change daocore library: add serialize and deserialize method for added PropertyType


Features
========

more than greenDAO, then exactly the same.

* support List<String>
* support Serialize object
* support Enum
* support add annotation for field
